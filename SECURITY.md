# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the `eodhd` SDK, please report it responsibly.

**Email:** [security@eodhistoricaldata.com](mailto:security@eodhistoricaldata.com)

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to release a fix within 7 days for critical issues.

## Scope

This policy covers the `eodhd` npm package source code. For vulnerabilities in the EODHD API itself, please contact [support@eodhistoricaldata.com](mailto:support@eodhistoricaldata.com).

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |
| < 1.0   | No        |
